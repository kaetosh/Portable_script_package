# Portable_script_package
Создание портативного комплекса скриптов на Python, который использует связку портативного терминала Windows и портативной версии Python. Позволяет запускать несколько скриптов в единой, удобной оболочке с современным интерфейсом, не требуя установки Python или настройки окружения на целевом компьютере.
